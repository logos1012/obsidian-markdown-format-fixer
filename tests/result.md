## **1.2 자주 묻는 질문 TOP 10**

**조사 기준: **Reddit, 공식 포럼, 국내 블로그/카페에서 가장 많이 반복되는 질문 분석

**Q1. 옵시디안을 처음 시작했는데, 뭐부터 해야 하나요?**

*핵심 니즈: *명확한 첫 걸음 가이드. 대부분의 입문자가 빈 Vault 앞에서 막막함을 호소함

*강의 대응: *5분 안에 첫 노트 작성하기 실습, 미니멀 시작 템플릿 제공

**Q2. 태그와 링크의 차이가 뭔가요? 언제 뭘 써야 해요?**

*핵심 니즈: *연결의 개념적 이해. #태그 vs [[링크]] vs 폴더 구분이 안 됨

*강의 대응: *실제 사례로 3가지 연결 방식 비교, 상황별 가이드라인 제시

**Q3. 폴더 구조를 어떻게 만들어야 하나요?**

*핵심 니즈: *"정답"에 대한 갈증. PARA, Johnny Decimal 등 방법론 중 뭘 써야 할지 고민

*강의 대응: *폴더는 나중에! 우선 노트를 쌓고 패턴이 보이면 정리하는 방식 안내

**Q4. 꼭 설치해야 하는 플러그인이 뭐예요?**

*핵심 니즈: *2000개+ 플러그인 중 필수템 선별. 정보 과부하로 분석 마비 상태

*강의 대응: *입문자용 미니멀 플러그인 5개만 추천 (Calendar, Dataview, Templater, Daily Notes, Tasks)

**Q5. 노션에서 넘어왔는데, 옵시디안이 너무 불편해요**

*핵심 니즈: *노션의 편의성(데이터베이스, 예쁜 UI)을 옵시디안에서 구현하고 싶음

*강의 대응: *노션과의 철학적 차이 설명 + 옵시디안만의 장점(로컬 저장, 링크 기반) 부각

**Q6. 마크다운을 꼭 배워야 하나요?**

*핵심 니즈: *코딩 같아 보이는 마크다운에 대한 두려움. 배우는 데 시간이 많이 걸릴 것 같은 부담

*강의 대응: *5개 필수 문법만 10분 안에 마스터 (제목, 굵게, 링크, 리스트, 체크박스)

**Q7. 핸드폰이랑 컴퓨터 동기화는 어떻게 해요?**

*핵심 니즈: *무료 동기화 방법. Obsidian Sync($96/년)는 비싸고, iCloud/Dropbox 설정이 복잡함

*강의 대응: *iCloud(iOS), Syncthing(Android) 단계별 설정 가이드

**Q8. 그래프 뷰가 예쁘게 안 나와요**

*핵심 니즈: *SNS에서 본 멋진 그래프 뷰를 재현하고 싶음. 내 그래프는 점 3개뿐

*강의 대응: *그래프 뷰는 목표가 아닌 결과물! 노트 100개 쌓인 후 자연스럽게 생성됨을 안내

**Q9. 데일리 노트를 어떻게 활용해야 하나요?**

*핵심 니즈: *단순 일기가 아닌 생산적 활용법. 매일 쓰는데 쌓이기만 하고 연결이 안 됨

*강의 대응: *데일리 노트 → 영구 노트 승급 프로세스, 실용적 템플릿 제공

**Q10. Dataview가 뭔가요? 꼭 써야 하나요?**

*핵심 니즈: *유튜브에서 많이 언급되는 Dataview에 대한 호기심. 코딩 같아서 두려움

*강의 대응: *기초 강의에서는 제외, "노트 50개 이상일 때 배우세요"라고 명확히 안내


# Part 2. 강의 목표 및 학습 성과 정의

## **2.1 수강생이 얻을 수 있는 구체적 결과 3가지**

**결과 1: 나만의 지식 시스템 구축**

- **측정 가능한 성과: **강의 완료 후 30개 이상의 연결된 노트 보유

- **기대 효과: **흩어진 메모를 한 곳에서 관리, 과거 기록을 즉시 검색 가능

- **마케팅 카피: **"더 이상 어디 적었는지 찾지 마세요. 모든 생각이 한 곳에."

**결과 2: 매일 쓰는 습관 정착**

- **측정 가능한 성과: **데일리 노트 루틴 정착 (강의 기간 중 14일 연속 작성)

- **기대 효과: **아침 5분 → 하루 계획, 저녁 5분 → 회고 습관화

- **마케팅 카피: **"하루 10분, 인생이 정리되는 경험"

**결과 3: 도구 스트레스 해방**

- **측정 가능한 성과: **5개 핵심 플러그인 설정 완료, 나만의 기본 세팅 확립

- **기대 효과: **"이 정도면 충분해"라는 확신, 더 이상 세팅 고민 X

- **마케팅 카피: **"세팅은 그만, 이제 활용에 집중하세요"

**Day 1 (시작): **옵시디안 설치 → 첫 노트 "나의 첫 노트" 작성 → "생각보다 쉽네?" 성취감

**Day 3 (적응): **데일리 노트 템플릿 설정 → 아침 루틴에 포함 → "이게 내 것이 되는 느낌"

**Day 7 (연결): **노트 간 [[링크]] 연결 시작 �� 과거 노트와 연결되는 순간 → "아! 이래서 옵시디안이구나"

**Day 14 (완료): **30개 노트 달성 → 그래프 뷰에서 연결망 확인 → "나만의 지식 네트워크가 생겼다!"
